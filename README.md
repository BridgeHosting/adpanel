# ⚙️ How to install?
**Just run this in your terminal:**

```bash
git clone https://github.com/portofoliox/adpanel.git && cd adpanel && bash initialize.sh && bash start.sh
```

# How to change password for Admin user or something else?
 **Just run this in your terminal:**
 
```bash initialize.sh
```
